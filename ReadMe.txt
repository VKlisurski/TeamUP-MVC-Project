TeamUp is individual work for course ASP.Net MVC at Telerik Academy.

TeamUp is application used for finding people you need for your football game, or finding a game if you want to join one.
All users  have ratings and you can choose the best for your game.

MVC application powered by:
	ASP.NET MVC - Repository Pattern + Unit of work
	KendoUI
	Microsoft SQL Server
	Twitter Bootstrap
	

This app requires KendoUI, UI for ASP.NET MVC. You can download it from here: http://www.telerik.com/download/kendo-ui
